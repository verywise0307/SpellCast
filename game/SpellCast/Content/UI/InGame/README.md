# InGame HUD 이미지

UMG에서 텍스트와 게이지 채움 이미지는 별도 위젯으로 올린다. 아래 PNG는 프레임과 장식 전용이다.

- `OpponentStatusFrame.png`: 상대 이름, 체력바, 라운드 표시 배경
- `PlayerStatusFrame.png`: 플레이어 초상화, 체력 및 마나 게이지 배경
- `SpellSlotFrame.png`: 활성 주문 카드 슬롯
- `WebcamFrame.png`: 카메라 영상 프레임
- `GesturePanel.png`: 인식 손동작과 시전 진행 표시 배경
- `CombatLogPanel.png`: 최근 전투 이벤트 목록 배경
- `ArcaneCrosshair.png`: 화면 중앙 조준점
- `DamageDirectionRight.png`: 오른쪽 피격 방향 표시
- `InGameHUD_Atlas.png`: 위 요소의 원본 아틀라스

`DamageDirectionRight.png`는 UMG에서 회전하여 다른 방향에도 재사용할 수 있다.
